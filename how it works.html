How It Works

1. Facial expression identification model based on movie recommendation system


In recent years, there has been a growing interest in utilizing facial expression recognition and analysis of social media behavior to create personalized movie recommendation systems. Recognizing this trend, the team at CoiCon proposed a novel approach to this problem by developing a movie recommender system that uses a combination of these two methods to determine a person's current mood and recommend movies that are well-suited to that mood.

2. Social media sentiment analysis model based on movie recommendation System


The proposed system employs advanced machine learning algorithms to recognize different facial expressions and analyze social media behavior. This information is then used to determine the user's current mood, which is subsequently used to recommend movies that are well-suited to that mood. The user interface is designed to be simple and intuitive, allowing users to easily navigate the system and find the perfect movie to watch.
